C++-ის შექმნა

C++ პროგრამირების ენა შეიქმნა 1979 წელს ბიარნე სტროუუსტრუპის მიერ. თავდაპირველად ის იყო C ენის გაფართოება და ეწოდებოდა "C with Classes". მისი მიზანი იყო ობიექტზე ორიენტირებული პროგრამირების დამატება C ენაში.

C++-ის განვითარება და სტანდარტები

წლების განმავლობაში C++ მნიშვნელოვნად განვითარდა. შეიქმნა სხვადასხვა სტანდარტები, როგორიცაა C++98, C++11, C++14, C++17 და C++20. თითოეულმა ახალმა ვერსიამ დაამატა ახალი შესაძლებლობები და გააუმჯობესა ენის ფუნქციონალი.

საინტერესო ფაქტი

C++ დღემდე ფართოდ გამოიყენება თამაშების, ოპერაციული სისტემების და მაღალი წარმადობის პროგრამების შესაქმნელად.
